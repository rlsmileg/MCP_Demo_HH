DatasauRus API

Static JSON endpoints generated from myDatasauRus.csv. These files can be hosted directly on GitHub Pages.

List datasets api/datasets.json Get x/y values by dataset api/datasets/dino.json

Example browser request:

const dataset = "dino";
const response = await fetch(`./api/datasets/${dataset}.json`);
const data = await response.json();

console.log(data.points);
